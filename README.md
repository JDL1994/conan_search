# Conan Package Search Tool

一个基于 Flask 的 Conan 包搜索工具,提供 Web 界面来搜索和下载 Conan 包。

## 功能特点

- 支持 Profile 名称的实时搜索和筛选
- 显示包的详细信息(Recipe ID、Package ID、Options、Settings)
- 提供包的下载链接
- 支持版本选择
- 美观的 Web 界面

## 技术栈

- 后端: Python Flask
- 前端: HTML, JavaScript, CSS
- UI 组件: Select2
- 包管理: Conan
